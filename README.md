# Netflix Movies Data Analysis Project

## Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a Netflix / movie dataset using Python.  
The main objective is to analyze movie ratings, popularity, and trends using data visualization techniques.

---

## Technologies Used
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## Dataset
- **File Name:** `mymoviedb.csv`
- The dataset includes movie-related information such as:
  - Movie Title
  - Ratings
  - Popularity
  - Vote Count

---

## Features of the Project
- Data cleaning and preprocessing  
- Handling missing values  
- Exploratory Data Analysis (EDA)  
- Data visualization using graphs and charts  
- Insight generation from data
